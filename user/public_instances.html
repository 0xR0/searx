Public Searx instances¶

Useful information¶

  • Up-to-date health report available on https://stats.searx.xyz 1, for onion (tor) services: https://stats.searx.xyz/tor.html

  • Searx instances mailing list & subscription page.

  • Some of the Searx instances have a CAcert SSL certificate. You can install the missing root cert from here.

  • To add your own Searx instance to this page send us your PR. A GitHub account is required to send PR or add an issue.

1

Note that most of the instances with a A+ grade in CSP column in this site are not fully functional - for example auto-completion may not work.
